1. Descripcion

La biblioteca NR48/49 usa el metodo de Newton Raphson para hallar la solucion de 
ecuaciones de una variable.

NR48/49 esta escrita en SysRPL, es de facil uso y proporciona si el usuario asi
lo requiere una tabla de iteraciones ( el programa que forma la tabla se llama
'Tabulador4849' y te lo puedes bajar completo de www.hpcalc.org ) o simplemente
el resultado.

2. Notas

- Si tu tabla o solución te da este resultado "/0" quiere decir que el programa
  termino por una división entre cero.
- Si te sale el mensage "Raiz No Hallada", puedes activar la bandera de usuario
  3 ( 3 SF ) para que te de igualmente el valor en el que paro las iteraciones
  el programa, para poder estimar asi un mejor valor inicial.
